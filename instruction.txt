Бот написан на Python 3.5
использовал библиотеку: pyTelegramBotAPI

Для начала нужно установить telegram на ПК и зарегистрироваться в мессенджере.
Найти в поиске @BotFather – это отец всех ботов в telegram, именно он их создаёт.
Пишем ему /newbot и отвечаем на два простых вопросов: имя бота и его username.
После чего @BotFather поздравит нас с успешным созданием бота и отправит нам его
token — 523870826:AAF0O8T-e7riRi8m6qlRz4pBKKdh0OfHKj8.

Внимание: token – единственный идентификационный ключ к боту.
Нигде не выкладывайте его, иначе другие люди смогут управлять Вашим ботом.
Бот с данным token на момент выкладывания статьи удалён.

// устанавливаем библиотеку для phyton:
$ pip3 install pytelegrambotapi

// в файле config.py прописываем свой токен и пользователей на которых не действуют запреты бота.

// запуск бота из католога командой:
$ python3 bot.py

//при добавлении бота в группу необходимо сделать его "администратором" и разрешить ему отпровлять и редактировать сообщения.
